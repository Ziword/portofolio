Daily Operational IT APP (DOIT)

This Android application was developed using Java programming language with MVVM design pattern. The backend of this application uses PHP with Laravel Lumen as its framework. This application was developed based on the request of the management of PT Merak Jaya Group, and it is used to monitor daily IT operational activities. The backend of this application is also connected to the CCTV cameras registered on the master to enable CCTV control.
