Merak Attendance Application (MANTAP)

The MANTAP (Merak Attendance Application) is an internal company application used by every employee at PT Merak Jaya Group to record their attendance. It is a direct request from the management of PT Merak Jaya Beton, used to replace fingerprint or face detection attendance machines. The application is built using Java programming language with an Android base, and utilizes Laravel Lumens backend with an Oracle database, as well as Node.js for face recognition. The application has location-based attendance restrictions, preventing users from checking in outside of the registered location. It also features a built-in face detection system for anti-spoofing and liveness detection. Additionally, the application has a server-side (backend) face recognition system utilizing Node.js and Tensorflow.
