Project IT (PRIT)

PRIT (Project Management Information Technology) is an Android-based project management application developed using the Java programming language. It follows the MVVM architectural pattern and is designed to manage ongoing projects in the IT department of the software engineering division of PT Merak Jaya Group. The application uses Laravel Lumen as its backend framework with Oracle DB as its database. It features a user-friendly interface and technical capabilities such as real-time project tracking, team collaboration, and issue management. The application also utilizes RESTful API to communicate with other external applications.
